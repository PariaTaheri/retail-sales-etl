# Retail Sales ETL Pipeline

## Project Overview
This project is an end-to-end data pipeline that extracts, cleans, and loads retail sales data.

## Tech Stack
- Python
- Pandas
- SQLite
- VS Code

## Pipeline Steps
1. Extract data from CSV
2. Clean data using Pandas
3. Save cleaned data
4. Load into SQLite database
5. Read and verify data
